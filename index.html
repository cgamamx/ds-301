Hypothesis Testing¶

statistics.inference.hypothesis.get_p_value(ha_parameter: float, distribution: str = 'norm', alternative: str = 'less', **kwargs) → float[source]¶

Calculates the p-value for a test

Parameters

  • ha_parameter – Value of the Alternate Hypothesis parameter

  • distribution – Theoretical distribution to use, currently it only supports ‘t’ or ‘norm’

  • alternative – Defines the alternative hypothesis. Possible values: ‘less’ (default), ‘greater’, or ‘two-sided’.

  • **kwargs – All the extra parameters will be packed in kwargs, i.e. df for degrees of freedom in the “t” dist

Returns

The calculated p-value

statistics.inference.hypothesis.single_proportion_test(sample: pandas.core.series.Series, category: str, p_0: float, alternative: str) → Dict[str, float][source]¶

Performs a single proportion test

Parameters

  • sample – Series with the count of two categorical variables. Check the example below for details.

  • category – The name of the category we want to use for the test.

  • p_0 – The proportion of the Null Hypothesis

  • alternative – Defines the alternative hypothesis. Possible values: ‘less’, ‘greater’, or ‘two-sided’.

Returns

Dict with the calculated “z” parameter and the p-value

Example

The following is an example of the format required for the sample parameter. The index values (yes, no) are the categories, and the values are the count of elements in each category:

>>> sample
Out[1]:
Relapse
no      4
yes    20
Name: Drug, dtype: int64
statistics.inference.hypothesis.two_mean_test(data_stats: pandas.core.frame.DataFrame, categories: Tuple[str, str], alternative: str, **args) → Dict[str, float][source]¶

Performs a two mean test

Parameters

  • data_stats – Summaary Statistics of two numerical variables. Check the example below for details.

  • categories – A Tuple with the name of the categories required to use the test. The order is important, the first element of the Tuple will be X1 and the second element will be X2. The Null Hypothesis is X1 - X2 = 0

  • alternative – ‘less’, ‘greater’, or ‘two-sided’.

  • **args – Optional parameters will be packed in args. Currently, there is only one optional parameter: df, which is used to specify how to calculate the degrees of freedom for the t-distribution. If the value of df is set to satterthwait, the Satterthwait approximation will be used. Any other values will result in the minimum of n1-1 or n2-1

Returns

Dict with the calculated “t” parameter and the p-value

Example

The following is an example of the format required for the data_stats parameter. The index values are the descriptive statistics of the numeric variable and it must include count, mean, and std. It can easily be generated with the pandas.DataFrame.describe method:

>>> data_stats
Out[1]:
Drink     Coffee        Tea
count  10.000000  11.000000
mean   17.700000  34.818182
std    16.693645  21.084678
...
max    52.000000  58.000000
statistics.inference.hypothesis.two_proportions_test(sample: pandas.core.series.Series, categories: Tuple[str, str], alternative: str) → Dict[str, float][source]¶

Performs a two proportions test

Parameters

  • sample – Series with the count of two categorical variables. Check the example below for details.

  • categories – A Tuple with the name of the categories required to use the test. The order is important, the first element of the Tuple will be used for p1 and the second element for p2. The Null Hypothesis is X1 - X2 = 0

  • alternative – ‘less’, ‘greater’, or ‘two-sided’.

Returns

Dict with the calculated “z” parameter and the p-value

Example

The following is an example of the format for the sample parameter. Similarly to single_proportion_test, the index values (Desipramine, Lithium, Placebo) are the categories, and the values are the count of elements in each category. The two_proportions_test, requires at least three categories:

>>> sample
Out[1]:
Drug
Desipramine    14
Lithium         6
Placebo         4
Name: Relapse, dtype: int64
